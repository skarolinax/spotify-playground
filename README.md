# Spotify IPOD

A small personal project inspired by the classic iPod interface, built with Spotify Web API integration and a custom iPod-style UI.

### Features

- Spotify authentication
- iPod-inspired navigation UI
- Music playback using Spotify Premium
- Responsive interface

## Environment Variables

Create a .env file in the root directory:

```bash
VITE_SPOTIFY_CLIENT_ID=your_client_id
VITE_SPOTIFY_TOKEN=your_token
```

## Run locally

First:

```bash
npm install
```

Then:

```bash

npm run dev

```

## Tech Stack

- React
- Vite
- Spotify Web API
- react-spotify-web-playback
